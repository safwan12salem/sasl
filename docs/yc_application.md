# Y Combinator Application - Sasl

## What is Sasl?
The first social network that works completely offline using mesh networking.
Users can post, chat, stream, sell, teach, and earn money without internet.

## Why now?
- 3.7 billion people lack reliable internet
- Existing social apps require constant connectivity
- WebRTC + Bluetooth are now universal in browsers
- Nobody has combined social media + mesh networking at scale

## What have you built?
- Full-featured social platform (40+ features)
- Working offline mesh protocol
- Marketplace, streaming, tutoring, gigs
- AI content tools
- 9 languages
- Deployed and functional
